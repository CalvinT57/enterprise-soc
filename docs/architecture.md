# Enterprise SOC Architecture

## Project Overview

This document defines the architecture for an enterprise Security Operations Center (SOC) environment designed to simulate a real-world corporate network.

The environment will include:

- Active Directory
- Windows Clients
- Linux Servers
- Wazuh SIEM
- Suricata IDS
- Sysmon
- Incident Response
- Threat Hunting
- Security Automation

This document will evolve throughout the project.
